# Card art

This folder is intentionally empty in the public repo. The app expects 52
PNGs here, one per card, named `<rank><suit>.png` using lowercase single
letters, e.g.:

```
as.png   # Ace of spades
10d.png  # Ten of diamonds
kh.png   # King of hearts
qc.png   # Queen of clubs
```

Ranks: `a 2 3 4 5 6 7 8 9 10 j q k`
Suits: `c` (clubs), `d` (diamonds), `h` (hearts), `s` (spades)

See the top-level [README](../../../README.md#card-art) for why these
aren't committed and where to get a set.
